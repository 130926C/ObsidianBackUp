#### 安装

* 官网 https://www.rust-lang.org
* Rust 安装
	* macOS，Linux，Unix-Like OS
		`curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`
	* windows在官网下载exe文件

Rust 基本命令
* `rustup update` 更新；
* `rustup self uninstall` 卸载；
* `rustup component add rustfmt` 添加组件；
* `rustup --version` 查看版本；
* `rustup show` 查看是稳定版还是夜版；
