这一篇文章介绍如何将代码推送到远程仓库，以及当遇到本地与远程冲突时如何处理。

---
## 推送代码到远程仓库

使用下面命令将代码推送到远程仓库：
```shell
$ git push <name>
```
* `name` 是一个可选项，但如果你在添加远程仓库链接时指定了 `shortname` 那么这个就必须提供，同时和你的 `shortname` 保持一致。

---
## 拉取远程仓库内容

使用下面命令将远程仓库中的内容拉取下来：
```shell
$ git pull
```
